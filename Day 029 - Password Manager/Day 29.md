## Notes

### .join()

```python
myTuple = ("John", "Peter", "Vicky")

x = "#".join(myTuple)

print(x)  # John#Peter#Vicky
```

### pyperclip package

```python
import pyperclip
pyperclip.copy('The text to be copied to the clipboard.')
pyperclip.paste() 'The text to be copied to the clipboard.'
```
